# Описание проекта
Финансовая компания, которая предоставляет льготные займы стартапам, хочет войти на инвестиционный рынок с прицелом на покупку, развитие и последующую перепродажу перспективных стартапов. Для разработки модели бизнеса нужна информация. В данном [проекте](https://github.com/ososovyan/data_analyst/blob/main/research_analysis_of_startups/research_analysis_startups.ipynb), чтобы выработать общее понимание, какие данные могут быть полезны, проводится исследование на исторических данных.
# Цели  задачи
**Цель**
Подготовить датасет и проверить, что данные в нём соответствуют здравому смыслу, а также ответить на вопросы как о предобработке, так и о значении данных для бизнеса.
**Задачи**
- Познакомится с данными и провести первичную предобработку
- Провести обработку данных и первичное исслеование основываясь на полученных данных
- Провести исследовательский анализ объединённых таблиц
- Дать заключительный вывод на основании проведенног исследования и рекомендации
# Инструменты
- python
- pandas
- matplotlib
- seaborn
