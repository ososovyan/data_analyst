# Выводы
- К общему стилю `snake_case` были приведены названия столбцов
- Далее, поскольку тип серии данных напрямую зависит от пропусков данных паралельно совершалось заполнение пропусков, а также приведение к нужному для работы типу Некоторые строки были удалены за неимением смысла, что-то заменили индикатором, что-то оставили без изменения (что привело к появлению новой категории без рейтинга на этапе категоризации).
- Был проведен глубокий анализ на наличие дубликатов как явных так и не явных. Было 16956 строк Стало 16711 строк Удалено 245 строк это 0,014 от первоначального количества менее двух процентов
- Далее отфильтровал данные в срезе нужных нам годов с 2000 по 2013
- Провел категоризацию не почетырем категориям высокая оценка, средня, низка, без оценки в результате чего появилось два новых поля отражающих категории для двух оценок 
- Последним этапом было составление топ-7 платформ по выпуску игр на данном временном интревале, первое место достается PS2 для которой было выпущено 2127 игр. Также очень много не оцененных игр, большая часть в основном игры имеют среедний рейтинг, очень мало выско оцененных игр, еще меньше нипзкооцененных
