# Описание
Этот [мини-проект](https://github.com/ososovyan/data_analyst/blob/main/hypothesis_testing_yandexknigi/hypothesis_testing.ipynb) состоит из двух частей. В первой части работа с данными Яндекс Книг и проверка гипотез. Во второй части проверены результаты A/B-тестирования для данных другого сервиса.
# Цели и задачи
Цель_1:проверить гипотезу, что пользователи из Санкт-Петербурга проводят в среднем больше времени за чтением и прослушиванием книг в приложении по сравнению с пользователями из Москвы
Цель_2:проверить гипоотезу, что упрощение интерфейса приведёт к тому, что в течение семи дней после регистрации в системе конверсия зарегистрированных пользователей в покупателей увеличится как минимум на три процентных пункта
# Иструменты
