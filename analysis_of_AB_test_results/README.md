# Описание проекта
Команда разработчиков рекомендательных систем создала новый алгоритм рекомендаций, который, по их мнению, будет показывать более интересный контент для каждого пользователя. В данном [проекте](https://github.com/ososovyan/data_analyst/blob/main/analysis_of_AB_test_results/analysis_AB_test_results.ipynb) расчтывакются параметры A/B-теста, который позволит проверить эту гипотезу, и проанализировать его результаты по его результатам.

# Цели  задачи
рассчитать параметры теста, оценить корректность его проведения и проанализировать результаты эксперимента.
# Инструменты
- python
- pandas
- matplotlib
- scipy
